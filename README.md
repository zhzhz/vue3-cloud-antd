# vue3-cloud-antd

本项目的目标是实现一款企业级工业设备管理系统
基于B/S模式开发

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
